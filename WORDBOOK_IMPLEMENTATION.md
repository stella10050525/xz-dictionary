### 1. 核心功能模块

#### 📥 词书上传与解析
- **支持格式**：
  - Excel (.xlsx / .xls) - 使用 SheetJS 库解析
  - CSV (.csv) - 原生 JavaScript 解析
  
- **数据要求**：
  - 第1列：单词（必填）
  - 第2列：词性（必填）
  - 第3列：中文释义（必填）

- **解析流程**：
  1. 选择文件上传

## 🔧 技术实现

### 依赖库
- **SheetJS (xlsx.js)**
  - CDN: https://cdn.jsdelivr.net/npm/xlsx@0.18.5/dist/xlsx.full.min.js
  - 用途：解析 Excel 文件
  - 自动加载，无需手动引入

### 浏览器支持
- Chrome/Edge (推荐)
- Firefox
- Safari
- 需要支持：
  - localStorage
  - IndexedDB
  - Web Speech API（朗读功能）
  - FileReader API

### 存储容量
- localStorage: 5-10MB（词书数据、进度）
- IndexedDB: 50MB-1GB（图片）
- 总计：足够存储数千个单词

## 🐛 已知限制

1. **文件大小**：单个词书建议不超过5000词
2. **图片大小**：单张图片限制10MB
3. **浏览器依赖**：数据存储在本地，清除浏览器数据会丢失
4. **跨设备同步**：不支持，需手动导出/导入
5. **例句质量**：自动生成的例句可能不够地道
