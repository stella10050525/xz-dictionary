# 备考词书使用说明

## 📚 功能介绍

备考词书功能允许您上传自己的单词表（Excel或CSV格式），系统会：
- 使用您上传的单词、词性、释义
- 自动生成肖战主题例句
- 支持自定义配图上传
- 记录学习进度

## 📋 文件格式要求

### Excel文件 (.xlsx / .xls)
第一行为表头（可选），从第二行开始为数据：

| 列号 | 内容 | 必填 | 示例 |
|------|------|------|------|
| A列（第1列）| 单词 | ✅ 必填 | love |
| B列（第2列）| 词性 | ✅ 必填 | v./n. |
| C列（第3列）| 中文释义 | ✅ 必填 | 爱；喜爱；爱情 |
| D列（第4列）| 备注 | ❌ 可选 | 高频词 |

### 多词性支持 🆕
系统支持一词多义的情况，使用 `/` 或 `;` 分隔：

**方式一：词性和释义一一对应**
```
单词      词性      释义
love      v./n.     爱；喜爱；爱情
support   v./n.     支持；拥护；支撑物
practice  v./n.     练习；实践；业务
```

**方式二：单一词性，多个释义**
```
单词      词性      释义
perfect   adj.      完美的；理想的；完全的
```

**方式三：多个词性，共用释义**
```
单词          词性          释义
professional  adj./n.       专业的；职业的
```

### CSV文件 (.csv)
第一行为表头（可选），从第二行开始为数据，逗号分隔：

```csv
单词,词性,释义,备注
love,v./n.,爱；喜爱；爱情,高频词
support,v./n.,支持；拥护；支撑物,多词性
handsome,adj.,英俊的；帅气的,四级词汇
```

## 🎯 使用步骤

1. **准备词书文件**
   - 按照上述格式准备Excel或CSV文件
   - 确保至少包含：单词、词性、释义三列

2. **上传词书**
   - 点击"📥 上传新词书"按钮
   - 选择文件（支持 .xlsx / .xls / .csv）
   - 等待解析完成

3. **学习单词**
   - 点击词书卡片进入学习
   - 点击单词查看详情（含肖战主题例句）
   - 自动生成例句，支持朗读
   - 可上传自定义配图

4. **标记进度**
   - 点击单词右侧的 ○ 按钮标记已学习
   - 已学习的单词显示 ✓ 标记
   - 自动统计学习进度

5. **收藏到生词本**
   - 在单词详情页点击 ⭐ 按钮
   - 添加到生词本方便复习

## 💡 注意事项

1. **文件大小限制**
   - 建议单个词书不超过 5000 个单词
   - 文件大小建议不超过 5MB

2. **数据格式**
   - Excel 文件请使用标准格式保存
   - CSV 文件使用 UTF-8 编码（避免中文乱码）
   - 单词必须是英文，释义必须是中文

3. **词性标注建议**
   - 使用标准缩写：n. / v. / adj. / adv. / prep. 等
   - 可以包含多个词性：n./v. 或 adj./adv.

4. **示例例句**
   - 系统会自动生成肖战相关例句
   - 例句包含肖战本人或其角色（魏无羡、顾一野等）
   - 所有例句都配有中英双语

## 🌟 特色功能

1. **肖战主题例句**
   - 自动生成与肖战或其角色相关的例句
   - 包含《陈情令》《王牌部队》等剧中角色
   - 让背单词变得更有趣

2. **自定义配图**
   - 每个单词支持上传一张图片
   - 图片存储在浏览器本地
   - 使用 IndexedDB，容量更大

3. **学习进度跟踪**
   - 自动记录已学习的单词
   - 实时显示学习进度百分比
   - 数据保存在本地浏览器

4. **生词本联动**
   - 可将词书中的单词添加到生词本
   - 统一管理所有收藏的单词

## 📊 数据管理

- **本地存储**：所有数据保存在浏览器本地
- **数据导出**：未来版本将支持进度导出
- **数据备份**：建议定期备份原始Excel文件
- **清除数据**：可删除单个词书，保留其他数据
