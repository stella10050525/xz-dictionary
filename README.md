# 🦐 XZ Dictionary - 小飞侠专属词典

一个为肖战粉丝打造的英语学习词典工具，让背单词变得更有趣！

## ✨ 功能特色

- 📚 **内置词书** - 小学、初中、高中英语核心词汇
- 🔥 **热词推荐** - 精选高频单词
- ⭐ **生词本** - 收藏难词，随时复习
- 📝 **专属例句** - 肖战主题例句，让学习更有趣
- 🎯 **连续学习** - 上一词/下一词快速切换
- 📊 **学习进度** - 实时追踪学习情况

## 🎓 词书内容

### 小学词库（518 词）
- 小学英语核心词汇
- 每个单词配有专属例句
- 支持多词性显示

### 例句特色
所有例句都以肖战及其角色为主题，例如：

> **after** (prep.)  
> Xiao Zhan greets fans after the performance.  
> 肖战演出之后和粉丝打招呼。

> **doctor** (n.)  
> Gu Wei and Liu Zhengliang are both doctors.  
> 顾魏和刘铮亮都是医生。

## 💡 使用方式

1. **查词** - 输入英文或中文查询
2. **浏览词书** - 按分类学习单词
3. **收藏生词** - 点击⭐添加到生词本
4. **连续学习** - 使用上一词/下一词按钮
