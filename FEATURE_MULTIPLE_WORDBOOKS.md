## 🔧 核心函数

### 1. uploadWordbook()
- 触发文件选择
- 解析Excel/CSV
- 显示自定义对话框

### 2. showWordbookCustomizeDialog(wordbook)
- 创建模态对话框
- 显示名称输入框
- 显示封面选择器
- 存储临时数据

### 3. selectCover(emoji)
- 高亮选中的封面
- 更新选择状态
- 视觉反馈（边框、背景、缩放、阴影）

### 4. confirmWordbookUpload()
- 验证名称输入
- 保存词书（含封面和名称）
- 刷新列表
- 显示成功提示

### 5. saveWordbook(wordbook)
- 生成唯一ID
- 设置默认封面（如未选择）
- 存储到localStorage
- 返回ID

## 🎯 使用流程

### 方式1：首次上传
1. 点击"📤 上传词书"
2. 选择Excel/CSV文件
3. 等待解析（显示加载动画）
4. 自定义词书名称
5. 选择封面图标
6. 点击"✅ 确定上传"
7. 查看词书卡片

### 方式2：上传多本
1. 在词书列表页
2. 再次点击"📤 上传词书"
3. 重复上传流程
4. 每本词书独立显示

### 方式3：管理词书
- **查看**：点击词书卡片
- **学习**：在词书详情页浏览单词
- **标记**：点击"○"标记已学
- **收藏**：点击"☆"加入生词本
- **删除**：点击"🗑️"删除词书
