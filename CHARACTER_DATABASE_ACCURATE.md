## 重要原则
1. 褒义词 → 围绕肖战本人,绝不出现其他明星
2. 中性/负面词 → 使用剧中角色,可以提及剧中其他角色(包括反派)
3. 例句必须符合中英文表达习惯,不硬套词义

## 角色列表

### 1. 《超星星学院》(2016) - 方天择
**类型**: 科幻/青春/校园/星座
**角色身份**: 超能星学院最大社团"合威会"会长,方氏家族继承人
**人设特征**: 
- 表面身份:狮子座继承人
- 真实身份:双子座(有分身能力)
- 性格:冷静、有领导力、重情义
- 关系:与万适存(同父异母兄弟)、程芝儿(女主)

**适用词汇**: 
- leader, president, society, organization, power
- duplicate, twin, secret, identity, heir, inheritance
- 学院、社团、领导、继承

**剧情关键词**: 十二星座、超能力、家族继承、身世之谜、学院争斗

**反派**: 方母(阴谋策划者)、蛇夫座(反派组织)

## 使用指南

### 褒义词处理
如: handsome, talented, dedicated, kind...
→ 使用肖战本人,不提任何剧集角色

### 中性词智能匹配
- 医疗相关 → 顾魏(外科医生) 或 刘铮亮(基层医生)
- 设计/建筑 → 盛阳(建筑设计师)
- 武术/练习 → 郭靖(勤学苦练) 或 魏无羡(剑术)
- 军事/战斗 → 顾一野(军人) 或 疾冲(战士)
- 探险/神秘 → 藏海(探险家)
- 间谍/秘密 → 任少白(地下工作者)
- 美食/烹饪 → 莫得闲(御膳总管)
- 创业/生意 → 肖春生(创业者)
- 修炼/魂力 → 唐三(魂师) 或 魏无羡(修仙)
- 领导/社团 → 方天择(会长)
- 命运/悲剧 → 时影(魔神)

### 负面词处理
可以使用剧中反派:
- 方母(阴谋)
- 蛇夫座(邪恶)
- 其他剧中反派角色

但绝不用于肖战本人!
